# Packages

The implementation workspace lives here.

## Shared Foundations

- `harness-core` — the common harness, task, and trace contracts.
- `sandbox-core` — the local policy sandbox and future sandbox adapters.
- `codex-core` — the first codex-style terminal-core reconstruction scaffold.

## Planned Waves

- `opencode-core` — client/server terminal architecture and plan/build agent split.
- `pi-core` — toolkit-first minimalism and composable local ergonomics.
- `omx-layer` — planning gates, persistent completion loops, and runtime state.
- `openagent-layer` — specialist orchestration, model routing, and aggressive parallel overlays.
- `openspec-layer` — artifact-driven execution and spec workflow integration.

The planned-wave packages intentionally start as placeholders so the monorepo structure is explicit before each implementation wave begins.
